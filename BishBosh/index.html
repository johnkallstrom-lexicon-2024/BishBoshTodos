Bish-Bosh 2.0

The bish-bosh will list all numbers between, for example 1 to 100, with some exceptions. If the number is equally divided with the bish-number then that number will be replaced with 'Bish'. If the number is equally divided with the bosh-number then that number will be replaced with 'Bosh'. If the number is equally divided with both, then that number will be replaced with 'Bish-Bosh'.
